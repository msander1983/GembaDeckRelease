# GembaDeck Release Notes

## v1.4.0
- Run external tools inline via `{#run "command"#}` template tags — output is captured and inserted into the expanded text
- Console window shown while tool runs so you can see the process executing
- Supports direct executables and shell commands with 30-second timeout

## v1.3.0
- Option to require Windows Hello authentication every time a secret is accessed (instead of once per session)
- New Security section in Settings tab

## v1.2.0
- Configurable global hotkeys for Add Shortcut and New from Clipboard
- Create new shortcuts directly from clipboard content
- Typing delay to prevent false shortcut triggers during fast typing
- Minimum two-character shortcut matching to avoid accidental expansions
- Start minimized to tray option
- Dock panel and app profile improvements
- Math expression evaluator for templates
- Fixed Windows Hello dialog appearing behind main window on multi-monitor setups
- Fixed overlay popup clipping at screen edges
- UI contrast and readability improvements

## v1.1.0
- Automatic 30-day free trial on first launch (no license key required to get started)
- Crash/error report handler with one-click email reporting
- Feedback button in the main window bottom bar
- Trial status indicator in version text

## v1.0.0
- Initial release
- Text expansions with template variables
- Macro/action step automation
- Dock panel with app profiles
- Windows Hello encrypted secrets
- License key activation
